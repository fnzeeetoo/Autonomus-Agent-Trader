# AgentTrader Dashboard

Next.js 14 dashboard with Recharts to display equity curve, trade blotter, and configuration.

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Data

The dashboard reads data from local JSON files generated by the backtesting/paper trading system:

- `data/paper_trades/equity_curve.json` – array of equity values over time
- `data/paper_trades/trades.json` – list of trade records

If these files do not exist, the dashboard will show placeholder data.

## Deployment to Vercel

```bash
vercel --prod
```

Set environment variable `NEXT_PUBLIC_APP_URL` if needed.

## Pages

- `/` – Equity curve chart (Recharts LineChart)
- `/trades` – Trade blotter table
- `/config` – Configuration form for strategies and risk limits (reads/writes `config/` YAML files)

## Notes

Configuration changes are written back to the filesystem; ensure proper permissions.
